class player

class player
{
public:
    uint64 ptr;
    vector3 origin;
    vector3 rotation;
    bool local;
    bool bot;
    bool scav;
    int health;
    int health_max;
    string name;
    int level;

    bounding_box bbox;

    vector3 bone[player_bone::bone_amt];
    collider body_collider;
    collider& collider[body_part_collider_type::collider_amt];
    string slot_item_id[7];
};

A class representing a player.

The ptr member is a pointer to the EFT.Player class instance.

The origin member represents the player's current position in the game world.

The local member is a boolean flag indicating whether the player is the local player (i.e. controlled by the current user).
The bot and scav members are boolean flags indicating whether the player is an AI-controlled bot or a scav, respectively.

The health member represents the current health of the player, while health_max represents the maximum health that the player can have.
The name member is a string representing the player's name.

The bbox member is an instance of the bounding_box class, which represents the bounding box of the player's model.
The bone member is an array of vector3 objects representing the positions of the bones in the player's model.
The body_collider member is an instance of the collider class representing the collision detection shape of the player's body.
The collider member is an array of references to collider objects representing the collision detection shapes for each of the player's body parts.

class market_item

class market_item
{
public:
    int avg_price;
    uint32 attributes;

    string get_name(int locale);
};

A class representing in-game market info for an item.

The avg_price member represents the average price for the item. The get_name() method returns the item's name in the specified locale. Available locales are 0 - english, 1 - russian.

The attributes member is a bitfield containing attributes. Possible attributes are:

enum market_item_attributes
{
    market_item_food = 1 << 0,
    market_item_meds = 1 << 1,
    market_item_weapon = 1 << 2,
    market_item_armored = 1 << 3,
    market_item_barter = 1 << 4,
    market_item_ammo = 1 << 5,
    market_item_backpack = 1 << 6,
    market_item_grenade = 1 << 7,
    market_item_helmet = 1 << 8,
    market_item_keys = 1 << 9,
    market_item_mods = 1 << 10,
    market_item_noflea = 1 << 11,
    market_item_rig = 1 << 12,
};

Please note that this enum is not defined in scripting.

class item

class item
{
public:
    uint64 ptr;
    vector3 origin;
    bool container;
    bool corpse;
    int stack_count;
    bool found_in_raid;

    int64 container_item_count;
    string container_item_id[];

    market_item& market_info;
};

A class representing a loot item, container or corpse.

The ptr member is a pointer to the appropriate class instance.

The origin member represents the item's current position in the game world.

The container and corpse members are boolean flags indicating whether the item is a container or a corpse, respectively.

The market_info member holds market information for the item.

class exfil_point

class item
{
public:
    uint64 ptr;
    vector3 origin;
    bool scav_point;
    collider point_collider;
    string name;
};

A class representing an exfiltration point.

The ptr member is a pointer to the appropriate class instance.

The origin member represents the exfiltration point's current position in the game world.

The scav_point member is a boolean flag indicating whether the exfiltration point is a scav point.

The point_collider member holds the exfiltration point's collider.

The name member holds the exfiltration point's name.
