class col_t

class col_t
{
public:
    // Constructors
    col_t(uint8 r, uint8 g, uint8 b, uint8 a);

    // Methods
    uint8& r();
    uint8& g();
    uint8& b();
    uint8& a();
};

A class representing a color with 4 channels (red, green, blue, and alpha).
The color channels can be accessed and modified using the r(), g(), b(), and a() member functions.
The col_t class can be constructed with the uint8 values of the color channels.

class vector3

class vector3
{
public:
    // Constructors
    vector3();
    vector3(float x, float y);
    vector3(float x, float y, float z);

    // Properties
    float x, y, z;

    // Methods
    float distance(vector3 other);
    float length();
    vector3 opSub(vector3);
    vector3 opAdd(vector3);
    vector3 opAdd(float);
    vector3 opMul(float);
};

A class representing a 3D vector, which has three floating-point components, x, y, and z.
The distance() method calculates the Euclidean distance between two vectors.
The length() method calculates the magnitude of the vector.
The opSub() and opAdd() methods overload the - and + operators, respectively, allowing vectors to be subtracted and added together.
The opAdd() method also has an overload that allows a vector to be added to a scalar.
The opMul() method overloads the * operator, allowing a vector to be multiplied by a scalar.

class quaternion

class quaternion
{
public:
    // Constructors
    quaternion();
    quaternion(float x, float y, float z, float w);

    // Properties
    float x, y, z, w;

    vector3 multiply(vector3 other);
};

A class representing a 4-dimensional vector with components x, y, z, and w.
The multiply() method multiplies the quaternion by a vector3 and returns the resulting vector3. This is a common operation in computer graphics for rotating a vector by a quaternion.

class bounding_box

class bounding_box
{
public:
    // Constructors
    bounding_box();

    // Properties
    int x, y, w, h;
    bool valid;
};

A class representing a 2-dimensional bounding box with properties x, y, w, h, and valid.
The x and y properties represent the top-left corner of the bounding box, while w and h represent the width and height, respectively.
The valid property indicates whether the bounding box is valid or not.

class collider

class collider
{
public:
    // Constructors
    collider();

    // Properties
    uint8 type;
    vector3 origin;
    vector3 center;

    vector3 size; // box
    float height; // capsule
    int direction; // capsule
    float radius; // sphere and capsule

    quaternion rotation;

A class representing a collider, which can be of different types: box, sphere or capsule. The collider has a size, which is a 3-dimensional vector for box colliders, a float height for capsule colliders, and a float radius for both sphere and capsule colliders. Capsule colliders also have a direction, which specifies the orientation of the capsule.
